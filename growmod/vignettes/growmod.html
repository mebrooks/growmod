Simulate data and graph it to get an idea of the general pattern.

t1 = c(`(Intercept)` = 1, age = 0.15, `I(age^2)` = -0.01, prevsize = 0.7, `prevsize:age` = 0, 
    sigma_proc = 0.01, time_growth_sd = 0.1, indiv_growth_sd = 0.1, indiv_age_growth_sd = 0, 
    indiv_cor = 0, size0_mu = 2.5, size0_sd = 0.5)

set.seed(1111)
sg = simobs(t1, nind = 500, ntime = 20, sigma_obs = 0.01, recap = 0.4)
ggplot(sg, aes(age, size)) + geom_point(alpha = 0.05) + geom_smooth() + theme_bw()

Fit some models of varying complexity.

m1=growmod(~1, ~1, data=sg)
m2=growmod(~age, ~1, data=sg)
m3=growmod(~age+I(age^2), ~1, data=sg)
m4=growmod(~age+I(age^2), ~age, data=sg)
m5=growmod(~age, ~age, data=sg)

Residuals

We can use the predict function to make predictions of the next size. These only include the fixed effects.

dat2=ddply(organize_data(sg, sigma_obs = 0)$Ldat$Predictors, ~ID, mutate,
    prevsize= size[match(age, age+1)]
)
dat2$pred=predict(m3, dat2)
dat2$residuals=dat2$pred-dat2$size
ggplot(dat2, aes(age, residuals))+geom_point()+geom_smooth() + theme_bw()
